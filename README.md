# Fintr

A financial tracking and analysis tool designed for personal and small business finance management.

## Overview

Fintr helps users track income, expenses, investments, and generate financial reports. It provides insights into spending patterns, budget performance, and financial health through intuitive dashboards and analytics.

## Features

- Income and expense tracking
- Budget creation and monitoring
- Investment portfolio tracking
- Financial reporting and analytics
- Bill payment reminders
- Multi-currency support
- Data export (CSV, Excel, PDF)
- Secure data encryption

## Tech Stack

- Frontend: React, TypeScript, Chart.js
- Backend: Python, Flask, SQLite
- Authentication: JWT, OAuth 2.0
- Deployment: Docker, Heroku
- Testing: Jest, PyTest

## Getting Started

1. Clone the repository
2. Install backend dependencies: `pip install -r requirements.txt`
3. Install frontend dependencies: `npm install`
4. Configure database and environment variables
5. Run the application: `docker-compose up` or separate backend/frontend

## Usage

Access the web interface at `http://localhost:5000` to start tracking your finances. Mobile-responsive design allows access from any device.
